== Description ==

Disable upgrade reminder is a plugin that disables the upgrade reminder at the top of the WordPress interface for all users except administrators. It is useful for any situation when you don't want anybody except admins worrying about available updates.

== Installation ==

1. Download, install. Done!

== Frequently Asked Questions ==

= Why can I still see the upgrade notification? =

If you are an admin, you will always see the notification. The notification is hidden only for subscribers, authors, editors and contributors
